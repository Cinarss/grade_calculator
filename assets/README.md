# Assets Directory

This directory contains all app assets including icons and splash screens.

## Required Assets

### Icons
- **icon.png** - Main app icon (1024x1024px)
- **adaptive-icon.png** - Android adaptive icon foreground (1024x1024px)
- **favicon.png** - Web favicon (48x48px)

### Splash Screens
- **splash.png** - Splash screen image (1284x2778px recommended)

## Asset Specifications

### App Icon (icon.png)
- **Size**: 1024x1024 pixels
- **Format**: PNG
- **Background**: Transparent or solid color
- **Content**: App logo/icon centered

### Android Adaptive Icon (adaptive-icon.png)
- **Size**: 1024x1024 pixels
- **Format**: PNG
- **Background**: Transparent (background color set in app.json)
- **Safe Zone**: Keep important content within 768x768px center area

### Splash Screen (splash.png)
- **Size**: 1284x2778 pixels (or 2048x2732 for iPad)
- **Format**: PNG
- **Background Color**: #6366f1 (set in app.json)
- **Content**: App logo centered

## Generating Assets

You can use online tools like:
- https://www.appicon.co/
- https://www.makeappicon.com/
- https://expo.dev/tools

Or create them manually using design tools like:
- Figma
- Adobe Illustrator
- Sketch
